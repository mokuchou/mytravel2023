Day 3 行程

==== 秋葉原 (考慮不去)


飯店 09:00
走路到 -> 都營-新宿線-馬喰橫山 -> 都營-新宿線-小川町(東京)
下車走路到 -> 東京地鐵-丸之內線-淡路町 -> 池袋

JR: 池袋 -> -> Animate池袋本店 -> JR: 池袋 -> JR: 涉谷

JR: 澀谷 -> 忠犬小八 -> 宮下公園 MIYASHITA PARK -> -> JR: 涉谷 -> JR: 東京

JR: 東京 -> 東京站一番街 -> 回家


東日本橋站 => 池袋 40min

都營-淺草線-日本橋站 -> 都營-淺草線-東日本橋站 20min

Sunshine City

Sunshine City分為四個主要大樓
World Import Mart大樓的2F -> ONE PIECE mugiwara store Ikebukuro
World Import Mart大樓的2F -> NAMJA TOWN 南夢宮主題樂園
World Import Mart大樓的 1F跟B1F(稱為Alta商場) 可以連通到隔壁的 王子大飯店大樓的 Alta美食街
王子大飯店大樓 的2F 有 Pokemon center
王子大飯店B1 為 Alta美食街, 有世界各地的咖啡
另外還有 陽光60大樓跟文化會館大樓

南夢宮主題樂園
(買門票, 不買設施, 昭和懷舊街道,歐風繪本奇幻世界,妖魔鬼怪番外地)
南夢宮餃子競技場網羅了日本各都道府縣最有名與最特殊的餃子，連日本餃子四天王名店都在這裡擺攤

PARCO 百貨

5F: 未來百貨概念店
6F: 任天堂旗艦店, Pokemon center, Jump Shop

東京車站

東京一番街
東京中央郵便局 (07:00~21:00), 動漫明信片
東京動漫人物街

JR東京站的 八重洲地下中央口 距離 東京站一番街 最近

退稅櫃台位在八重州北口的右手邊, 一樓的星巴克旁, 達5000日圓就可以到櫃台辦理退稅
東京站一番街在買東西之前, 先到櫃台辦理免稅登記, 出示護照後會拿到免稅導覽手冊、免稅申請表, 退稅的店家只限紅底白字的退稅標誌
商店結帳時, 將免稅申請表交給店員, 店員便會將收據貼在申請表, 全部購物完後要再回到退稅櫃台, 出示免稅申請表、購買商品。

TOKYO GIFT PALETTE: 田謹製 紅豆餡奶油麵包屋(紅豆內餡奶油費南雪), CLUB HARIE e-challenge (年輪蛋糕),
PISTA＆TOKYO(開心果巧克力夾心餅乾、開心果奶油夾心餅乾)
東京動漫人物街: 常設型商店像是寶可夢, 角落生物, 蠟筆小新, 拉拉熊, 正能量企鵝, 光之美少女等人氣動漫主題專賣店
東京芭娜娜專賣店: 東京芭娜娜咖哩麵包, 東京芭娜娜的奶油紅豆甜甜圈

東京拉麵街, 日本美食街道, 東京甜點樂園
